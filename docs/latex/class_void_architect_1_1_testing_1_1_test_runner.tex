\doxysubsection{Detailed Description}
Simple test runner for \doxylink{namespace_void_architect}{Void\+Architect} engine components. 

\doxylink{class_void_architect_1_1_testing_1_1_test_runner}{Test\+Runner} provides a lightweight framework for registering and executing unit tests throughout the engine. Tests are automatically discovered and can be run individually or as a complete suite.

Key features\+:
\begin{DoxyItemize}
\item Automatic test registration using macros
\item Individual or batch test execution
\item Timing and performance measurement
\item CMake integration ready
\item CI/\+CD friendly output formatting
\end{DoxyItemize}
