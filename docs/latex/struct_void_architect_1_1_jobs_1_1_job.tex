\doxysubsection{Detailed Description}
Core job structure containing execution data and metadata. 

\doxylink{struct_void_architect_1_1_jobs_1_1_job}{Job} represents a unit of work that can be executed by the job system. \doxylink{namespace_void_architect_1_1_jobs}{Jobs} support dependency management through atomic counters, priority-\/based scheduling, and comprehensive state tracking for debugging and profiling.

Memory layout is optimized for cache efficiency with hot data (state, dependencies) placed at the beginning of the structure.
