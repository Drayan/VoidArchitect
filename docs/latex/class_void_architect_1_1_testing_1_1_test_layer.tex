\doxysubsection{Detailed Description}
\doxylink{class_void_architect_1_1_layer}{Layer} that executes the test suite. 

\doxylink{class_void_architect_1_1_testing_1_1_test_layer}{Test\+Layer} integrates with \doxylink{namespace_void_architect}{Void\+Architect}\textquotesingle{}s layer system to run tests at the appropriated time during the application lifecycle. Tests are executed once during the first fixed update, then the application is signaled to shut down gracefully.
