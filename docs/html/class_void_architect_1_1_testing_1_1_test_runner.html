Detailed Description

Simple test runner for VoidArchitect engine components.

TestRunner provides a lightweight framework for registering and executing unit tests throughout the engine. Tests are automatically discovered and can be run individually or as a complete suite.

Key features:

  • Automatic test registration using macros
  • Individual or batch test execution
  • Timing and performance measurement
  • CMake integration ready
  • CI/CD friendly output formatting

Usage example:

// Register a test (usually at file scope)
VA_REGISTER_TEST(MyComponentBasics, []()
{
MyComponent comp;
return comp.IsValid() && comp.DoSomething();
});
// Run all tests
int main()
{
return TestRunner::RunAllTests();
}
