Detailed Description

Result structure returned by job execution.

JobResult provides a standardized way to report job completion status and error information. This enables robust error handling and debugging throughout the job system.

Usage examples:

// Successful job
return JobResult::Success();
// Failed job with error message
return JobResult::Failed("File not found: config.json");
// Cancelled job
return JobResult::Cancelled("User requested shutdown");
