Macro to register a test function with the test runner.

Parameters
nameUnique test name (will be stringified)
funcFunction or lambda that returns bool (true = pass, false = fail)

This macro should be used at file scope to register tests. The test will be automatically discovered and included in test runs.

Example usage:

VA_REGISTER_TEST(CollectionsBasics, []()
{
VAArray<int> arr;
arr.push_back(42);
return arr.size() == 1 && arr[0] == 42;
});
