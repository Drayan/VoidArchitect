Detailed Description

Core job structure containing execution data and metadata.

Job represents a unit of work that can be executed by the job system. Jobs support dependency management through atomic counters, priority-based scheduling, and comprehensive state tracking for debugging and profiling.

Memory layout is optimized for cache efficiency with hot data (state, dependencies) placed at the beginning of the structure.
